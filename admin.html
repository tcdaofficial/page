GitHub Pages (ฟรี):
1. สร้าง Repository ใหม่บน GitHub
2. อัพโหลด index.html + admin.html + logo_text.jpg
3. ไปที่ Settings → Pages → เลือก main branch
4. เว็บจะออนไลน์ที่ https://username.github.io/repo-name

เชื่อม Domain thaicda.org:
ไปที่ Settings → Pages → Custom domain → ใส่ thaicda.org
